## Short-term (2026)

### Q1-Q2: First Working Plugin
- Take one real pain point from daily BIM coordination work
- Build a plugin that solves it
- Show colleagues, get feedback
- This becomes first "real" development experience

### Q2-Q3: Portfolio
- 2-3 plugins on GitHub with proper README + screenshots
- Video demos (1-2 min each)
- Polish DSTools to "showable" state

### Q3-Q4: Transition
- **Option A:** Propose automation role within R1
- **Option B:** Apply for "BIM Developer" / "Revit Developer" positions
- Unique selling point: Coordinator experience + development skills

**Target salary:** 100,000 — 180,000 ₽
